# Linksapp

![screen_recording](https://user-images.githubusercontent.com/19251998/198886868-0bd8affb-8f0c-445b-b31e-0e122697c751.gif)

### Usage

Adjust `profile.json` accordingly and start the project:

```
deno task start
```

or deploy using Deno Deploy.

### Features

- Social links
- Github readme
- Banner
- External links
- RSS feed
